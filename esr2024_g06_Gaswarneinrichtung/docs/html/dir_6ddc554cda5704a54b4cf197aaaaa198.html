Dateien

 AccelerationHandler.c
 This Class implements the IAccelerationHandler.h header file. It uses Pin 9 (P1.3) and 10 (P1.2) for the I2C connection, Pin 31 (P3.2) for activating/deactivating the sensor and Pin 29 (P3.5) for the digital value input.
 
 GasHandler.c
 This Class implements the IGasHandler.h header file. It uses Pin 32 (P3.7) for activating the Gas Sensor and Pin 30 (P3.1) for the digital input value or Pin 25 (P5.1) for the analog input value.
 
 I2CHandler.c
 This Class implements the I2CHandler.h header file.
 
 OutputHandler.c
 This Class implements the IOutputHandler.h header file. It uses the two onboard LEDs of the MSP430FR2355 as well as a Piezo speaker on Pin 19 (P2.0).
